# 轻想纪念版

把那些年写下的连载，留在自己手里。

用 **[轻想连载导出工具](https://github.com/thatgameapple/lianzai-export)** 备份数据后，用这个 app 在本地重温。

---

## 功能

- 浏览所有连载，封面卡片展示
- 完整阅读每篇内容与评论
- **随机回忆**：随机抽取一篇，意外重逢
- **那年今日**：找出历年同一天写下的内容

## 下载

前往 [Releases](https://github.com/thatgameapple/lianzai-reader/releases) 下载：

- **macOS**：`.dmg` 文件
- **Windows**：`.zip` 文件，解压后运行

## 使用方法

1. 先用导出工具备份你的轻想连载数据
2. 打开轻想纪念版，将备份文件夹拖入窗口，或点击「打开备份文件夹」
3. 开始回忆
